# CS2 Sticker Investment Tracker

Live price tracking dashboard for CS2 Major sticker investments. Fetches prices from the Steam Market, tracks performance over time, and deploys as a static site via GitHub Pages.

## Create Your Own Tracker

Track your own sticker portfolio in under 5 minutes.

### 1. Fork & Clone

Fork this repository, then clone your fork:

```bash
git clone https://github.com/YOUR_USERNAME/budapest2025.git
cd budapest2025
```

### 2. Import Your Inventory

Make sure [Bun](https://bun.sh) is installed, then run the import script with your Steam profile URL or vanity name:

```bash
bun import-inventory.ts YOUR_STEAM_URL
```

Examples:
```bash
bun import-inventory.ts oldm8clint
bun import-inventory.ts https://steamcommunity.com/id/oldm8clint/
bun import-inventory.ts 76561198012345678
```

This creates a `stickers.json` file with your inventory. The tracker will automatically load it.

**For a different event:**
```bash
bun import-inventory.ts YOUR_STEAM --event "Shanghai 2024" --cost 0.39
```

**Private inventory?** See [Manual Entry](#manual-entry) below.

### 3. Configure

Edit `config.json` with your details:

```json
{
  "event": "Budapest 2025",
  "costPerUnit": 0.35,
  "currency": "AUD",
  "currencyCode": 21,
  "currencySymbol": "$",
  "steamProfile": {
    "vanityUrl": "YOUR_STEAM_VANITY",
    "displayName": "YOUR_NAME",
    "avatarUrl": "https://avatars.akamai.steamstatic.com/YOUR_AVATAR_full.jpg"
  },
  "discord": { "enabled": false }
}
```

**Currency codes:** 1 = USD, 2 = GBP, 3 = EUR, 21 = AUD, 20 = CAD — [full list](https://partner.steamgames.com/doc/store/pricing/currencies)

**Finding your avatar URL:** Visit your Steam profile, right-click your avatar image, and copy the image URL.

### 4. Test Locally

```bash
bun tracker.ts
```

This fetches current prices and generates `index.html`. Open it in your browser to preview.

### 5. Deploy to GitHub Pages

1. Go to your fork's **Settings > Pages**
2. Under "Source", select **GitHub Actions**
3. Push your changes:

```bash
git add stickers.json config.json
git commit -m "Add my sticker inventory"
git push
```

The included GitHub Actions workflow (`.github/workflows/update-prices.yml`) will:
- Run `bun tracker.ts` every 15 minutes
- Commit updated `index.html` and price history
- Deploy to GitHub Pages automatically

Your tracker will be live at `https://YOUR_USERNAME.github.io/budapest2025/`

### 6. Updating Your Inventory

Re-run the import script anytime you buy or sell stickers:

```bash
bun import-inventory.ts YOUR_STEAM_URL
git add stickers.json
git commit -m "Update inventory"
git push
```

## Manual Entry

If your Steam inventory is private, create `stickers.json` manually:

```json
{
  "stickers": [
    { "name": "NiKo", "quality": "Normal", "qty": 10 },
    { "name": "NiKo", "quality": "Embroidered", "qty": 2 },
    { "name": "FaZe Clan", "quality": "Holo", "qty": 1 },
    { "name": "Legends", "quality": "Capsule", "qty": 20 }
  ],
  "event": "Budapest 2025",
  "totalQty": 33
}
```

**Quality values:** `Normal`, `Embroidered`, `Gold`, `Holo`, `Normal (Champion)`, `Embroidered (Champion)`, `Holo (Champion)`, `Capsule`

**Sticker names** must match Steam Market names exactly. Check the [Steam Market](https://steamcommunity.com/market/search?appid=730&q=Sticker+Budapest+2025) for correct spelling.

## Discord Notifications (Optional)

To get Discord alerts for price changes, milestones, and portfolio updates:

1. Create webhook URLs in your Discord server (Server Settings > Integrations > Webhooks)
2. Add them to `config.json`:

```json
{
  "discord": {
    "enabled": true,
    "webhooks": {
      "alerts": "https://discord.com/api/webhooks/...",
      "portfolio": "https://discord.com/api/webhooks/...",
      "milestones": "https://discord.com/api/webhooks/..."
    }
  }
}
```

Or set them as GitHub Actions secrets (recommended for public repos):
- `DISCORD_ALERTS`, `DISCORD_PORTFOLIO`, `DISCORD_MILESTONES`, `DISCORD_SIGNALS`, `DISCORD_WEEKLY`, `DISCORD_SKINPORT`, `DISCORD_MAJORS`

## How It Works

- **tracker.ts** — Main script. Fetches Steam Market prices, Skinport data, exchange rates, and generates the HTML dashboard with charts and analytics.
- **track-majors.ts** — Tracks historical major sticker prices for prediction models. Runs every 2 days.
- **import-inventory.ts** — Imports your CS2 sticker inventory from Steam into `stickers.json`.
- **config.json** — Your personal configuration (currency, profile, event, Discord).
- **stickers.json** — Your sticker inventory (auto-generated by import script).
